Customizing your product.

Stokke is a Norwegian manufacturer of children's furniture and accessories. They focus on producing high quality products for babies and children.

Easy interface

We have partnered with North Kingdom to build them a configurator to allow customers to customize some of their well known products. The configurator features a minimalistic interface to let you focus on the product.

Mobile friendly

There is a a strong focus on the mobile experience - one of the main requirements for the project. Everything should feel fluid and with a snappy interaction response.

Site integration

Everything is built with the back-end integration in mind. The app sits on top of the website framework and is connected with the check-out process. The same application can be re-used to display different products.
