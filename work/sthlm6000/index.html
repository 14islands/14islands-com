Improving student housing in Stockholm.

Sthlm6000+ is an urban development project that aims to build more than 6000 student apartments in Stockholm.

Building it

The project brings together a large number of players in Stockholm; municipalities, construction companies, educational institutions and students.

Our job was to bring designs to life with interactivity and animations.

The goal of the site is to present this project in a good way and provide information to the public. The site is built responsive in Wordpress to make it fully editable by the client.

Design

We were approached by BVD to build the website for this project. BVD is an experienced design studio in Stockholm that focuses on simplicity and quality for their clients.

They showed us their design concept for the site, it featured bold imagery from Stockholm and nice typography. It had us at hello.

Laying the grounds

All content on the site is editable using Wordpress as the Content Management System. One of the best things about Wordpress is that it provides such a pleasant experience for the editors that manage all the different types of content for the site.

The admin interface in Wordpress is responsive, making it possible to manage the site on a wide variety of devices. This is just one of the reasons we like to use Wordpress.
