{% markdown %} ## Everyday struggle The idea was to give people the feeling of how it’s to live with Parkinson. Everyday tasks can become a struggle, such as switching a lightbulb or put a tooth paste on your brush. The game makes precision pretty difficult as vibrations come break the tasks the user needs to accomplish. {% endmarkdown %}

{% markdown %} ## Swish to support At the end of the game, people can donate the result of the game to support this worthy cause. We made it seamless by integrating with Swish - the number one app in Sweden to transfer money. {% endmarkdown %}

At the core we built a responsive web site using standard HTML, JavaScript and CSS. The 3D world is rendered on a canvas inside this framework using WebGl. {% endmarkdown %}
