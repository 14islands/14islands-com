{% markdown %} ## Easy interface We have partnered with North Kingdom to build them a configurator to allow customers to customize some of their well known products. The configurator features a minimalistic interface to let you focus on the product. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/stokke/stokke-ui.png" alt="Stokke configurator" ratio="16_10" margin="top" %}
{% markdown %} ## Mobile friendly There is a a strong focus on the mobile experience - one of the main requirements for the project. Everything should feel fluid and with a snappy interaction response. {% endmarkdown %}
{% include casestudy-video-inline.html vimeoId="217012318" ratio="16_10" %}
{% markdown %} ## Site integration Everything is built with the back-end integration in mind. The app sits on top of the website framework and is connected with the check-out process. The same application can be re-used to display different products. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/stokke/stokke-stroller-choices.jpg" alt="Stokke configurators" ratio="16_10" margin="top" %}
