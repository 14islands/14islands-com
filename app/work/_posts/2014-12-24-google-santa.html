{% markdown %} ## 'Tis the Season For the second year in a row, we helped build Google's Santa Tracker experience. We've had the pleasure to build 16 interactive scenes for this massive christmas environment with lots of fun animations. Each day - from December 1st to December 24th - we get to follow Santa and his elves as they prepare for Christmas. {% endmarkdown %}

{% markdown %} ## Decorations The overall experience was directed by our friends at [UPPERQUAD](http://upperquad.com) and [UENO](http://ueno.co). The design features colourful flat elements such as moving animals and elves in different colour outfits. All design elements are vector based - a good approach to keep things flexible and resolution independent. As always, we made everything responsive to make it look great on all kinds of different devices. This year we even support Chromecast and Android Wear integration. {% endmarkdown %}

{% markdown %} ## Santa's factory As always, when working with Google, we got to play with a lot of new toys. This year Santa's factory was upgraded to use Web Components (using Polymer) and the new Web Animations specification. Read more about our thoughts on Web Animations on the blog. * [Polymer](https://www.polymer-project.org/), Web Components polyfill * [Web Animations](https://github.com/web-animations/web-animations-js), Animations polyfill * [Sass](http://sass-lang.com/), CSS pre-processing. * [Modernizr](http://modernizr.com/), Browser and device feature detection. * [jQuery](http://jquery.com/), DOM manipulation and more. * [Gulp](http://gruntjs.com/), Task Automation. * [Bower](http://gruntjs.com/), Front-end package manager. {% endmarkdown %}
