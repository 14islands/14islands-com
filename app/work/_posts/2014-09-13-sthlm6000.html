{% markdown %} ## Building it The project brings together a large number of players in Stockholm; municipalities, construction companies, educational institutions and students. Our job was to bring designs to life with interactivity and animations. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/sthlm6000/home-hero-second-16-9.jpg" alt="Home page" margin="both" ratio="16_9" %}
{% markdown %} The goal of the site is to present this project in a good way and provide information to the public. The site is built responsive in Wordpress to make it fully editable by the client. {% endmarkdown %}
{% include casestudy-responsive.html url="/images/work/sthlm6000/sthlm6000-iphone6-shortened.jpg" parallax=true parallaxOffset="-0.6" parallaxDirection="vertical" %}
{% markdown %} ## Design We were approached by [BVD](http://bvd.se) to build the website for this project. BVD is an experienced design studio in Stockholm that focuses on simplicity and quality for their clients. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/sthlm6000/home-hero-16-9.jpg" alt="Home page" margin="both" ratio="16_9" %}
{% markdown %} They showed us their design concept for the site, it featured bold imagery from Stockholm and nice typography. It had us at hello. {% endmarkdown %}

{% markdown %} ## Laying the grounds All content on the site is editable using Wordpress as the Content Management System. One of the best things about Wordpress is that it provides such a pleasant experience for the editors that manage all the different types of content for the site. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/sthlm6000/projekt-bostad-16-9.jpg" alt="Map with project locations" margin="both" ratio="16_9" %}
{% markdown %} The editor in Wordpress is responsive, making it possible to manage the site on a wide variety of devices. This is just one of the reasons we like to use Wordpress. {% endmarkdown %}
