{% markdown %} ## A fantastic journey Together with North Kingdom we created an immersive world consisting of 4 individual games - each one focusing on different aspect of online safety. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/google-interland/mindful-mountain.jpg" alt="Interland - intro" ratio="16_10" margin="both" %}
{% markdown %} This project has been a long and fantastic journey for us and we were happy to bring many creative solutions to the table. It included multiple runs of user tests to make sure the games were fun and understandable for young kids. {% endmarkdown %}
{% markdown %} ## Interactive story telling The inhabitants of Interland give life and personality to each part of the Interland story. The blue Internaut is the protagonist of the story and is with us throughout the experience. {% endmarkdown %}
{% include work/google-interland/internaut.html %}
{% markdown %} Each game also has an antagonist, or Blabh as we like to call them in Interland, which the player needs to overcome. {% endmarkdown %}
{% include work/google-interland/shouter.html %}
{% markdown %} The characters all have a number of animations that are triggered based on what happens around them, making them feel more life-like. {% endmarkdown %}
{% markdown %} ## Bully-free An important part of Interland is the ability to use it in schools, where hardware often consists of tablets or Chromebooks with relatively small screens and less powerful CPUs. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/google-interland/interland-tablet.png" alt="Interland - intro" ratio="16_10" margin="top" %}
{% markdown %} We take great pride in the fact that the whole experience runs smooth even on older mobile phones. {% endmarkdown %}
{% markdown %} ## Be Internet Awesome Interland is part of the “Be Interland Awesome” initiative by Google, to teach kids about digital citizenship and online safety. On top of Interland, Google asked for a site to explain the overall project for parents and people that are interested this worthy cause. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/google-interland/InterlandHub-screenshot_16_9.jpg" alt="The Interland Hub" ratio="16_9" margin="both" %}
{% markdown %} We delivered the companion site as part of the Interland framework --- focusing on providing quality content in a friendly, informative and responsive manner. {% endmarkdown %}
{% markdown %} ## Creative Tech This was a complex project that required many tools and a combination of creative technologies. At the core we built a responsive web site using standard HTML, JavaScript and CSS. The 3D world is rendered on a canvas inside this framework using WebGl. {% endmarkdown %}
