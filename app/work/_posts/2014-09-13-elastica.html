{% markdown %} ## Building the hype Key points of this section: 14islands created this because we are awesome. Goal is to spread the Elastica brand. It's a fast, responsive website. Full of engaging animations and transitions. Important to build confidence - visitors must trust Elastica. {% endmarkdown %}

{% markdown %} ## Design This is one of many successful projects we've done together with [UENO](http://ueno.co). The Elastica website features bold colors, long storylike pages and creative animations. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/elastica/elastica-cloudsoc-desktop-2_1.png" alt="Elastica threats page" margin="top" ratio="2_1" %}
{% markdown %} We absolutely love taking on designs with this level of creativity and ambition. It makes our hard work and attention to every last detail so much more rewarding. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/elastica/elastica-blog-desktop-2_1.png" alt="The Elastica blog with large engaging hero images" margin="both" ratio="2_1" %}
{% markdown %} A blog was built in Wordpress which features large cover images and styled content. All text and images are completely dynamic and editable — empowering the editors to create new, beautifully designed pages. {% endmarkdown %}
{% markdown %} ## Bringing it to life In addition to crafting a fully responsive experience, the goal was to create an exciting story as you scroll through the pages. {% endmarkdown %}

See the Pen Spinning Satellites by 14islands (@14islands) on CodePen.

{% markdown %} We created tasteful animations and transitions to enhance the content and guide visitors through the website. {% endmarkdown %}

See the Pen Circular mask by 14islands (@14islands) on CodePen.

{% markdown %} Animations run based on the scroll position and plays when entering the viewport. {% endmarkdown %}

See the Pen Dancing Circles by 14islands (@14islands) on CodePen.

{% markdown %} All in all, we ended up handcrafting over 20 animations, and we had a lot of fun doing it. {% endmarkdown %}
