{% markdown %} ## Launching early In the spirit of moving fast, our first iteration was to build a static site using HTML, CSS & JavaScript. This meant they could go live early, and the result was a blazing fast responsive website, with engaging transitions and animations. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/sthlm6000/home-hero-16-9.jpg" alt="Home page" margin="both" ratio="16_9" %}
{% markdown %} We then circled back and converted high priority pages into Wordpress templates — giving the Elastica marketing team the control they needed to make fast changes. {% endmarkdown %}
{% include casestudy-responsive.html url="/images/work/sthlm6000/home-mobile.jpg" %}
{% markdown %} ## Design This is one of many successful projects we've done together with [UENO](http://ueno.co). The Elastica website features bold colors, long storylike pages and creative animations. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/sthlm6000/about-pie-chart-16-9.jpg" alt="About page with pie chart" margin="both" ratio="16_9" %}
{% markdown %} We absolutely love taking on designs with this level of creativity and ambition. It makes our hard work and attention to every last detail so much more rewarding. {% endmarkdown %}
{% markdown %} ## Bringing it to life In addition to crafting a fully responsive experience, the goal was to create an exciting story as you scroll through the pages. We ended up handcrafting over 20 animations which all run based on the scroll position. {% endmarkdown %}
