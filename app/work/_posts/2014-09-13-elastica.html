{% markdown %} ## Building the hype We set out on a mission to combine Responsive Web Design, captivating animations and great performance — realising how important it is for a company like Elastica to build confidence in their visitors. {% endmarkdown %}
{%comment%} {% include wide-content-image.html src="/images/work/elastica/elastica-threats-desktop-2_1-8bit.png" alt="Elastica threats page" margin="both" ratio="2_1" %} {%endcomment%}
{% markdown %} The site was partially built in Wordpress, in order to give the Elastica marketing team the control and freedom they need to work with the content. {% endmarkdown %}
{% include casestudy-responsive.html url="/images/work/elastica/elastica-iphone6-full-page-shortened-8bit.png" parallax=true parallaxOffset="-0.5" parallaxDirection="vertical" %}
{% markdown %} ## Design & UX This is one of many successful projects we've done together with [UENO](http://ueno.co). The Elastica website features bold colors, long storylike pages and creative animations. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/elastica/elastica-cloudsoc-desktop-2_1.jpg" alt="Elastica threats page" margin="top" ratio="2_1" %}
{% markdown %} We absolutely love taking on designs with this level of creativity and ambition. It makes hard work and attention to detail so much more rewarding. {% endmarkdown %}
{% include wide-content-image.html src="/images/work/elastica/elastica-blog-desktop-2_1.jpg" alt="The Elastica blog with large engaging hero images" margin="both" ratio="2_1" %}
{% markdown %} A blog was built in Wordpress which features large cover images and styled content. All text and images are completely dynamic and editable — empowering the editors to create new, beautifully designed pages. {% endmarkdown %}
{% markdown %} ## Bringing it to life In addition to crafting a fully responsive experience, the goal was to create an exciting story as you scroll through the pages. We built tasteful animations and transitions to enhance the content and guide visitors through the website. {% endmarkdown %}
{% include work/elastica/spinning-satellites.html %}
{% markdown %} We used a mix of techniques ranging from CSS3 animations, JavaScript animations and HTML5 Canvas rendering to get the right result. {% endmarkdown %}
{% comment %} {% include casestudy-codepen.html slug="znLKm" ratio="16_9" %} {% endcomment %}
{% markdown %} Animations on the Elastica website run based on the scroll position, and they are timed to play exactly when entering the viewport. {% endmarkdown %}
{% comment %} {% include casestudy-codepen.html slug="poydt" ratio="16_9" %} {% endcomment %}
{% markdown %} All in all, we ended up handcrafting over 20 animations, and we had a lot of fun doing it. {% endmarkdown %}
