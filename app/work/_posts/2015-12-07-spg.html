{% markdown %} ## Enter the Experience The idea with the website is to present SPG and their real estate portfolio in a clear and inspiring way. Simple structure, interactive elements and smooth transitions between pages make the user experience seamless. The website supports multiple languages and is currently available in Swedish ([spg.se](http://spg.se){:target="_blank"}) and Norwegian ([spg.no](http://spg.no){:target="_blank"}). It's made with WordPress so its fully editable by the people at SPG, and its built responsive to work well on mobile, tablets and desktop computers. {% endmarkdown %}
{% include casestudy-responsive.html url="/images/work/spg/spg-iphone6-full-page-shortened-8bit.png" parallax=true parallaxOffset="-0.6" parallaxDirection="vertical" %}
{% markdown %} ## Design [BVD](http://bvd.se) designed the identity and website. BVD is an experienced design studio in Stockholm that focuses on simplicity and quality for their clients. {% endmarkdown %}
