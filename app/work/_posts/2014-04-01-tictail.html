{% markdown %} ## About [Tictail](http://www.tictail.com/) is a Stockholm-based startup that enables people that create products to easily sell online. They wanted a modern website for the Tictail brand, increase signups and to help their community to succeed. The site experience is both fun, fast and responsive. It was also built in Wordpress to make content fully editable by the people at Tictail. {% endmarkdown %}

{% markdown %} ## Technologies The external technologies used for this project. * [WordPress](https://wordpress.org/), Content management. * [Advanced Custom Fields](http://www.advancedcustomfields.com/), Multible content types for WordPress. * [SEO by Yoast](http://www.advancedcustomfields.com/), SEO optimization for WordPress. * [GridsetApp](https://gridsetapp.com/), Responsive Grid Systems. * [responsive.io](https://responsive.io/), Responsive Images. * [Sass](http://sass-lang.com/), CSS pre-processing. * [Modernizr](http://modernizr.com/), Browser and device feature detection. * [ScrollMonitor](https://github.com/sakabako/scrollMonitor), Scroll detection. * [jQuery](http://jquery.com/), DOM manipulation and more. * [FastClick](https://github.com/ftlabs/fastclick), Eliminate 300ms delay for touch events. * [Grunt](http://gruntjs.com/), Task Automation. * [Bower](http://gruntjs.com/), Front-end package manager. {% endmarkdown %}
